Un outil de collaboration

Créer une image

Il est facile de créer une image de VM à partir de la machine actuelle...

vagrant package --base web --output web.box

... et de la réutiliser


Vagrant.configure(2) do |config|
    config.vm.box = "web"
    config.vm.box_url = "file://web.box"
end

Donner un accès distant à son application

Sur votre poste:

vagrant share --http 80

De l'autre côté:

vagrant connect SHARE_NAME

Publier son image


config.push.define "ftp" do |push|
    push.host = "ftp.xebia.fr"
end
        
vagrant push web

Exercice

Packagez un répertoire contenant les serveurs web et db, et le Vagrantfile permetttant de démarrer de façon autonome.
