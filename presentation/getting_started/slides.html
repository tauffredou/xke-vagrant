Ma première VM

(Le provider sera VirtualBox)

Création d'une VM Ubuntu trusty 64 bits avec une configuration par défaut

vagrant init ubuntu/trusty64

Lister les machines Vagrant existantes

vagrant global-status

 id       name           provider   state    directory
 -----------------------------------------------------------------------------------
 f9c8e17  default        virtualbox running  /Users/thomas/xke/vagrant

Connexion en ssh à la VM

vagrant ssh

Eteindre la VM

vagrant halt

Recharger la VM

vagrant reload

Supprimer la VM

vagrant destroy

Créez une machine sous ubuntu et installer nginx
