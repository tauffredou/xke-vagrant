Multi machines

Vagrant permet de démarrer une architeture complète


Vagrant.configure("2") do |config|

    config.vm.box = "ubuntu/trusty64"

    config.vm.define "web" do |web|
        web.vm.box = "nginx"
        web.vm.network :private_network, ip: "192.168.13.37"
    end

    config.vm.define "db" do |db|
        db.vm.box = "redis"
        db.vm.network :private_network, ip: "192.168.13.38"
    end
end

Démarrer les deux machines

vagrant up

Démarrer une machine en particulier

vagrant up db

Accès SSH à une machine en particulier

vagrant ssh db
